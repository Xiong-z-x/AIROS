# Open Source Go2W Reference Meshes

These meshes are copied from `ypat999/3d_dog_navi_ros2` and are used only by
the optional `robot_visual_profile:=reference_mesh` display profile.

- Source repository: <https://github.com/ypat999/3d_dog_navi_ros2>
- Source path:
  `src/ignition_models/gazebo_garden_migration/models/go2w/meshes/`
- License in source repository: Academic Free License 3.0
- License copy in this repository:
  `docs/third_party_3d_dog_navi_ros2_AFL-3.0_LICENSE`

AIROS keeps the already verified analytic collision geometry and wheel-control
interfaces. These meshes improve visual fidelity without importing the full
Gazebo Garden / CHAMP / low-level legged controller stack.
